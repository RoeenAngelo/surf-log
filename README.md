# Surf Log 🏄

## Description
A CLI app to record and track surf sessions.
Tracks surf conditions, time of day, crowd factor, 
and equipment used. Useful for choosing a spot and 
equipment given the day's forecast.

## Features
- Add and view surf spots
- Log surf sessions with conditions and equipment
- Delete spots and sessions
- Rate sessions out of 10
- View stats including total sessions and average rating
- Data persists between sessions via JSON

## How to run
1. Make sure you have Python 3.13.3 installed
2. Clone the repo: `git clone https://github.com/RoeenAngelo/surf-log.git`
3. Navigate to the folder: `cd surf-log`
4. Run the app: `python surf_log.py`

## Tech used
